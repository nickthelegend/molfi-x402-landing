# Molfi Landing Website

Polished marketing site introducing Molfi.fun's hybrid payment rails and showing off our product vision to the hackathon judges.

## Quickstart

1. **Install dependencies**:
   ```bash
   pnpm install
   ```
2. **Run dev server**:
   ```bash
   pnpm run dev
   ```

## Acceptance Checklist

- [ ] Displays beautiful hero section, diagram of the x402 transaction flow, and models cost table.
- [ ] Responsive design with a Lighthouse mobile score target of ≥ 95.
- [ ] Contains all hackathon submission details, badges, and demo links.
